# X::Real::Ip

Replace REMOTE_IP with X-Real-Ip if it's trusted (useful for Nginx)

## Installation

Add this line to your application's Gemfile:

    gem 'x-real-ip'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install x-real-ip

## Usage

For rails just add it to Gemfile
